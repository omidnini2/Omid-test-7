# Persian Voice Cloning AI Web App

یک وبسایت رایگان برای کلون کردن صدای کاربر و تبدیل متن فارسی به گفتار با صدای کلون‌شده.

## ویژگی‌ها
- تم لایت/دارک
- آپلود و ضبط صدا (تا ۲۰ دقیقه و ۵۰ مگابایت)
- نوشتن متن (تا ۱۰۰,۰۰۰ کاراکتر)
- دانلود و اشتراک‌گذاری خروجی
- پشتیبانی از زبان فارسی و سایر زبان‌ها
- اتصال به مدل هوش مصنوعی رایگان (so-vits-svc یا RVC)

## راه‌اندازی سریع
1. سایت را روی XAMPP/WAMP اجرا کنید (فایل‌های public را در htdocs قرار دهید).
2. مدل هوش مصنوعی را طبق آموزش راه‌اندازی کنید (در پوشه docs).
3. سایت را به مدل متصل کنید و از کلون صدا و TTS فارسی لذت ببرید!

---

A free website for Persian voice cloning and TTS with user voice.

## Features
- Light/Dark theme
- Upload/record voice (up to 20min, 50MB)
- Input text (up to 100,000 chars)
- Download/share output
- Persian and multi-language support
- Connects to free AI model (so-vits-svc or RVC)

## Quick Start
1. Run the site on XAMPP/WAMP (put public files in htdocs).
2. Setup the AI model (see docs folder).
3. Connect the site to the model and enjoy Persian voice cloning and TTS!
